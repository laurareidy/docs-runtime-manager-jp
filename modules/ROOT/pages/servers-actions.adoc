= サーバのシャットダウン、再起動、削除方法 (ハイブリッド)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

ハイブリッドデプロイメントシナリオでは、サーバを作成した後、Runtime Manager からサーバの状況を制御することができます。

. *[Servers (サーバ)]* ページで、制御するサーバを選択します。
. 情報パネルで、サーバ状況の表示ラベルをクリックし、ドロップダウンリストから実行するアクションを選択します。次のアクションを実行できます。
+
* *Shutdown (シャットダウン)*
* *Restart (再起動)*
* *Delete*
* *Renew Certificate (証明書を更新)*

. 選択したアクションを実行することを確認します。

サーバをシャットダウンした後に Runtime Manager から再起動することはできません。Mule Runtime Engine がインストールされているマシンで、サーバを手動で再起動する必要があります。

サーバのシャットダウンまたは再起動中にエラーが発生した場合、Mule Runtime Engine がインストールされているマシンで、アクションを手動で実行する必要があります。  このシステムへのアクセス権があることを確認してください。
