= サーバグループの作成 (ハイブリッド)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]


. Anypoint Platform にサインインし、*[Runtime Manager]* をクリックします。
. 左側のメニューで *[Servers (サーバ)]* をクリックしてから *[Create Group (グループを作成)]*​をクリックします。
. ​*[Create Server Group (サーバグループを作成)]*​ページで、サーバグループの名前を入力し、新しいサーバグループに含めるサーバを選択してから、​*[Create Group (グループを作成)]*​をクリックします。
+
[TIP]
グループ名は、先頭と末尾がダッシュではなく、3 文字以上 40 文字以下の一意の名前で、文字、数字、ダッシュのみを含む必要があります。

新しいサーバグループが *[Servers (サーバ)]*​リストに表示されます。サーバグループ名をクリックして、グループに属するサーバのリストを表示します。
