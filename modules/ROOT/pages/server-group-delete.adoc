サーバグループを削除する場合、次の点に留意してください。

* サーバグループのアラート定義と履歴は削除されます。
* サーバグループに割り当てられているサーバは削除されません。削除されたグループに属するサーバはメインサーバリストに表示さます。これらのサーバは個別に使用するか、他のサーバグループやクラスタと組み合わせることができます。
* グループに割り当てられたサーバは再起動またはシャットダウンされません。
* サーバグループ内のサーバにデプロイされたアプリケーションは各サーバでコピーされます。


== サーバグループの削除

. Runtime Manager の *[Servers (サーバ)]* タブでサーバグループエントリをクリックすると、画面の右側に詳細ビューが表示されます。

. サーバ名の下にある下矢印をクリックし、*[Delete group (グループを削除)]* をクリックします。

+
image::deletegroup.png[DeleteGroup]

. 続行の確認を促すメッセージが表示されます。チェックボックスをクリックし、*[Delete Server Group (サーバグループを削除)]* をクリックします。
